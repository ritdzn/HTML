Introdução do HTML na prática

Para iniciarmos no HTML é importante termos duas coisas instalas em seu PC

  1. Visual Studio Code
  2. Plug-in Live server no VS CODE

Padrões usadas no desenvolvimento WEB

Sempre devemos criar o nosso arquivo HTML com o titulo "INDEX.HTML"

HTML é composto basicamento de 3 partes:

  1. HTML

    2. Tudo o que está na página deve estar dentro da tah HTML

  2. HEAD

    3. Tudo o que carrega antes de aparecer o site para o usúario

  3. BODY

    4. Tudo o que aparece de conteúdo para o usúario
